# decode2019sample

de:code2019 MVPパーソナルスポンサー サンプルコードです。

## 動作環境
RaspberryPi3(Raspbian Stretch 04/08)
Tensorflow 1.13
OpenCV4.1.0
Python3.6

## 使い方
サンプルに含まれるモデルを利用する場合にはリポジトリをRaspberryPiにコピーして、
コマンドラインにて「python3 predict.py [画像ファイル]」を実行してください。
